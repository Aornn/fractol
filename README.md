# Fract'Ol

How to use:

 * arrow to move
 * "+" and "-" to modify delta translation 
 * Wheel to zoom and zoom out
 * For Julia you need to lock with "L" if you want to move or zoom, and "U" is for unlock
 * R = reset

Instruction :+1:
```shell
git clone https://github.com/Aornn/fractol.git
```
```shell
make
```
```shell
./fractol julia or mandelbrot or tricorn
```
